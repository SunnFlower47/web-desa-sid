## Tentang Proyek
Cibatu Vibe AI merupakan aplikasi frontend berbasis Next.js 14 yang bertindak sebagai antarmuka publik bagi warga Desa Cibatu. Aplikasi ini terhubung langsung ke Backend Laravel Admin Panel via Proxy Middleware internal Next.js demi keamanan.
