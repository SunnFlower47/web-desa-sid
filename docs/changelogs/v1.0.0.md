# Changelog v1.0.0

**Tanggal Rilis:** 12 Juni 2026

## Fitur Baru (New Features)
- **Halaman Beranda (Landing Page):** Implementasi Hero Slider modern, statistik cepat, fitur unggulan, dan portal berita terintegrasi.
- **Layanan Warga:** Modul Bantuan Sosial (Cek NIK), Pengajuan Surat, dan Pengaduan Masyarakat terpadu.
- **Informasi Transparansi:** Dashboard visualisasi ringkas untuk APBDes, Proyek Desa, dan Bantuan Sosial Transparansi.
- **Sistem Testimoni:** Form interaktif bagi warga untuk memberikan *feedback* publik dengan validasi keamanan.

## Keamanan (Security)
- Integrasi **Google reCAPTCHA v2** pada form Testimoni (Beranda & halaman khusus), Kontak, dan pencarian Bantuan Sosial untuk mencegah *bot spam*.
- Implementasi sistem Proxy Internal Next.js API Routes untuk menyembunyikan API key dan URL asli backend Laravel dari inspeksi *client-side*.

## Deployment (Ops)
- Optimalisasi Dockerfile untuk mode `standalone` build dari Next.js.
- Penyertaan file `.env` ke dalam runtime Docker menggunakan COPY directive, guna memastikan variabel OS Server-Side terbaca oleh *container* Google Cloud Run.
- Konfigurasi file `.gcloudignore` spesifik untuk menjaga kebersihan proses *upload source code*.
